Multi-Sensor Fusion

Fitur Gabungan Paling Canggih

01
Composite Stress Score (CSS)
EDA (35%) + HRV inverse (35%) + HR zone (30%) → skor stres terpadu 0–100
02
TRIMP (Training Impulse)
Beban latihan harian: HR zone × durasi × faktor personal. Lacak progres fitness mingguan.
03
VO₂max Estimate
15 × (MHR / HRrest) — estimasi kapasitas aerobik dari sensor HR.
04
Lactate Threshold Detection
HR drift + cadence drop + EDA spike → "Mendekati ambang laktat" secara real-time.
05
Heat Stress Index
Core temp + HR% + suhu lingkungan → risiko heatstroke, peringatan dini.
06
Dehydration Risk Signal
EDA drop + HR naik + hydration level → "Minum sekarang" sebelum terlambat.
07
Recovery Readiness (1–10)
HRV pagi + HR istirahat + suhu inti → skor kesiapan latihan hari ini.
08
Neuromuscular Fatigue Index
GCT naik + cadence drop + asimetri → "Kurangi pace, otot kelelahan."
09
Geospatial Env-Physio Fusion
GPS pilih otomatis stasiun IQAir/BMKG terdekat → PM2.5 + UV + temp difusi real-time dengan data fisio → rekomendasi Go / Moderate / Stay Indoors.
10
Respiratory Stress Index (RSI)
PM2.5 (40%) + HR% (40%) + SpO₂ dip (20%) → risiko gangguan pernapasan 0–100.
