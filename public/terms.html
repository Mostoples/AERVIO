Versi 1.0 · 29 Mei 2026

Dengan menggunakan AERVINEX, Anda setuju dengan syarat berikut. Baca sebelum melanjutkan.

1. Definisi

"AERVINEX" merujuk pada aplikasi mobile + smartwatch yang dioperasikan oleh penelitian skripsi independen. "Layanan" mencakup semua fitur app + cloud sync + ML inference.

2. Lisensi Penggunaan

Kami memberi Anda lisensi non-eksklusif untuk menggunakan AERVINEX sesuai dengan kebijakan ini. Anda TIDAK boleh: reverse-engineer, jual data Anda ke pihak ketiga via API kami, atau menggunakan layanan untuk tujuan ilegal.

3. Bukan Pengganti Konsultasi Medis

AERVINEX adalah tool skrining wearable berbasis riset. Hasil tidak menggantikan diagnosis dari tenaga medis berlisensi. Selalu konsultasi dokter untuk kondisi yang Anda alami.

4. Disclaimer Risiko

  • ML predictions adalah estimasi statistik, bukan kepastian klinis
  • Sensor onboard ada toleransi error (±0.1°C suhu, ±2% SpO₂, ±10% PM2.5)
  • Alert L3 (kritis) wajib direspons — tidak menerima alert bukan jaminan aman
  • Untuk gejala akut: panggil 119 (BSMI/Ambulans) atau ke IGD terdekat

5. Akun & Keamanan

  • Anda bertanggung jawab atas keamanan password akun
  • Anda harus berusia 13 tahun atau lebih untuk membuat akun
  • Kami berhak menonaktifkan akun yang melanggar kebijakan tanpa pemberitahuan

6. Konten Pengguna

Data sensor + assessment Anda tetap milik Anda. Dengan opt-in riset agregat, Anda memberi lisensi non-eksklusif untuk pemakaian data anonymized dalam publikasi akademis.

7. Pembatasan Layanan

  • Firebase Spark plan: 50K Firestore reads/hari, 20K writes
  • BMKG API: rate-limit 10K calls/hari (shared)
  • BLE inference: latency tergantung quality koneksi

8. Perubahan Layanan

Fitur dapat ditambah, diubah, atau dihapus dengan notifikasi 7-30 hari (tergantung kritikalitas).

9. Limit of Liability

Sebagai project riset akademis non-komersial, kami tidak bertanggung jawab atas kerugian langsung/tidak langsung dari penggunaan AERVINEX. Layanan disediakan "as-is".
