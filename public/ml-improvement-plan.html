15 Features untuk Port (formula matematis)

Time-domain HRV RMSSD, SDNN, pNN50, CoV(RR), MAD-RR
SDNN AF threshold > 100 ms · CoV AF > 0.10 Frequency-domain (Welch / Lomb-Scargle) LF (0.04-0.15Hz), HF (0.15-0.40Hz), LF/HF ratio Poincaré + Non-linear (best AF discriminator) SD1 = sqrt(0.5) · SD(RR_{i+1} − RR_i)
SD2 = sqrt(2 · SDNN² − 0.5 · SDSD²)
SD1/SD2 ratio · SampEn (m=2, r=0.2·SD) · Shannon dRR
TPR (turning point ratio) · COSEn = SampEn + ln(2r) − ln(mean(RR))
AF threshold COSEn > −1.4 (Lake & Moorman 2011) Signal quality gates (mandatory per DeepBeat) PPG-SQI ≥ 0.6, mean(RR) ∈ [0.3, 2.0] s

Arsitektur Target: XGBoost → ONNX → onnxruntime-web

Bundle ~50-200 KB, target AUC 0.90-0.93, F1 0.70-0.78, Acc 88-91%. 15 features di-flatten → tree budget <200 trees depth ≤6. Pipeline: train Python offline (PhysioNet 2017 [D1] + Icentia11k [D2]) → onnxmltools.convert → fetch+cache via Service Worker → ort.InferenceSession.

Closed-Form Formulas (siap port ke JS)

Uth VO₂max (15.3 coefficient verified) VO₂max_ml_kg_min = 15.3 × (HRmax / HRrest)
HRmax = 208 − 0.7 × age (Tanaka 2001, lebih akurat dari 220-age)
HRrest = window terendah 5-min rolling, pre-activity Cole HR Recovery HRR_1min = HR_peak − HR_60s_post_exercise
HRR_3min = HR_peak − HR_180s_post_exercise
risk_flag = HRR_1min < 12 // 4× mortality hazard
healthy = HRR_1min ≥ 18 AND HRR_3min ≥ 42 Plews Lognormal RMSSD Baseline lnRMSSD_t = ln(RMSSD_t)
mu_7d, sigma_7d = mean/std lnRMSSD last 7 days
z_today = (lnRMSSD_today − mu_7d) / sigma_7d
flag_overtraining = z_today < −1.0 sustained >3 hari
SWC = 0.5 · sigma_7d // per-individu Gabbett ACWR (sweet spot, BUKAN threshold) acute = sum(load over last 7d)
chronic = mean(weekly_load over last 28d)
ACWR = acute / chronic
sweet_spot: 0.8 ≤ ACWR ≤ 1.3
danger: ACWR > 1.5 Stull 2011 Wet-bulb approximation Twb ≈ T·atan(0.151977·sqrt(RH+8.313659))
+ atan(T+RH) − atan(RH−1.676331)
+ 0.00391838·RH^1.5·atan(0.023101·RH) − 4.686035
WBGT_outdoor = 0.7·Twb + 0.2·Tg + 0.1·Tdb Buller EKF Core Temp (parameter dari paper) State: CT_k (core temp) · Observation: HR_k
a = −7.379, b = 384.4, c = −4881
Q = 0.000484 (process variance)
R = 18.88 (observation variance)
Init: CT_0 = 37.1°C, P_0 = 0, update tiap 60s
HR_hat = a·CT² + b·CT + c
H = 2·a·CT + b (Jacobian)
Critical: CT ≥ 39.5°C = heat illness flag Hew-Butler 2015 EAH Composite weight_change_pct = (post − pre) / pre × 100
EAH_risk = weight_change_pct > 0
EAH_high = weight_change_pct > +2
dehydration = weight_change_pct < −2
Composite race ≥4h: + fluid > 800 mL/h + sodium < 300 mg/h → HIGH Periard Heat Acclimatization Timeline heat_accl_days = jumlah hari dalam 14d dengan WBGT_max >28 AND exercise >30 min
status: naive (<5) · partial (5-9) · full (≥10)

Arsitektur Target: HYBRID (closed-form first, ML second)

Bukti literatur kuat mendukung formula deterministik untuk 4 label ini. Memaksa ML black-box meniru Uth/Buller/Plews akan kalah akurat selama dataset training masih semi-synthetic. ML hanya untuk multi-label co-occurrence ranking — prioritise risk ketika beberapa formula trigger bersamaan.

Re-train cv-fitness dengan UMA Treadmill (992 tests) akan langsung naikkan F1 ≥0.80 dari 0.72. Per-user lognormal baseline wajib untuk overtraining; absolute threshold akan selalu salah karena variance inter-individu.
