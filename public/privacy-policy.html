Kebijakan Privasi

Versi 1.0 · 29 Mei 2026

AERVINEX menghormati privasi pengguna. Kebijakan ini menjelaskan apa yang kami kumpulkan, mengapa, dan bagaimana Anda mengontrolnya.

1. Data yang Dikumpulkan

1.1 Data Akun

  • Nama, email, foto profil (jika login Google)
  • Profil kesehatan: umur, BB, TB, gender, target aktivitas

1.2 Data Sensor (dari smartwatch)

  • HR, HRV, SpO₂, respiratory rate, temperatur kulit, hidrasi
  • PM2.5, UV index, heat index dari sensor onboard + stasiun terdekat
  • Aktivitas: langkah, kalori, sesi olahraga

1.3 Data Lokasi

  • GPS untuk: rute lari, pilihan stasiun BMKG/IQAir terdekat
  • Tidak dijual ke advertiser, tidak dipakai untuk tracking

2. Bagaimana Kami Pakai Data

  • ML inference: TEPRS, AFib screening, Recovery score, dst.
  • Alert kontekstual: peringatan PM2.5 spike, heatstroke risk, dehidrasi
  • Riset agregat (opt-in): data anonymized untuk publikasi akademis
  • TIDAK: dijual ke pihak ketiga, dipakai untuk advertising, atau diserahkan tanpa surat resmi hukum

3. Penyimpanan & Keamanan

  • Firestore region asia-southeast2 (Jakarta)
  • TLS in transit · AES-256 at rest (Firebase default)
  • Authentication: Firebase Auth (Google, email/password, anonymous)
  • Retention: data sensor 90 hari di server, lifetime di device (opt-out anytime)

4. Hak Anda (UU PDP No. 27/2022 & GDPR Art. 15–22)

  • Akses — lihat seluruh data pribadi yang kami simpan via Profile → Privasi → Activity
  • Koreksi — perbarui profil & data kesehatan via Profile → Edit Profil
  • Portabilitas — 📥 export semua data dalam format CSV/JSON yang machine-readable
  • Penghapusan — 🗑️ hapus akun + seluruh data terkait secara permanen (irreversible). Selesai < 30 hari.
  • Pembatasan pemrosesan — 🔄 opt-out riset agregat & ML training data kapan saja
  • Keberatan — tolak pemrosesan untuk tujuan tertentu (mis. analytics) via Settings → Privasi
  • Audit log — 🔍 lihat siapa & kapan mengakses data Anda

Untuk eksekusi hak di atas, gunakan menu in-app atau email dpo@aervinex.app. Kami merespon dalam 7×24 jam kerja.

5. Bukan Diagnostik Medis

AERVINEX adalah tool skrining wearable, BUKAN alat diagnostik. Untuk gejala signifikan, konsultasi dokter atau panggil 119.

6. Anak di Bawah 13 Tahun

AERVINEX tidak ditujukan untuk anak <13. Mayoritas model ML kami belum validated pada populasi pediatrik.
