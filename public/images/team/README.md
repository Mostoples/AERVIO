## Spesifikasi rekomendasi
- **Format**: JPG (quality 85) atau WebP
- **Ukuran**: 400×400 px square (rasio 1:1)
- **Background**: solid color atau soft gradient — akan di-clip menjadi circle di UI
- **Lighting**: portrait headshot, mata sejajar kamera
- **File size**: ≤ 80 KB per foto

Setelah upload, foto otomatis terpakai di landing page section Team. Kalau file tidak ada, fallback ke gradient initial avatar (huruf besar).

Update file ini setiap nambah foto baru.
