09 · Honest Reporting

Limitasi · Bukan untuk Diagnosis Medis

⚠️ Disclaimer

AERVINEX adalah tool skrining wearable, BUKAN alat diagnostik medis. Semua angka risk factor adalah indikator dari sensor + ML, bukan pengganti pemeriksaan klinis. Bila Anda mengalami gejala signifikan, konsultasi dokter atau panggil 119 untuk darurat.

Limitasi Utama

  • Population: Mayoritas dataset training berasal dari populasi non-Indonesia. Kalibrasi TUHAM (Indonesia ISPU + Riskesdas) dilakukan tapi belum tervalidasi multicentre lokal.
  • Pediatric: Mayoritas model belum validated pada anak <12 tahun. Threshold dewasa di-apply secara umum — bukan presisi pediatri.
  • Sensor calibration: Akurasi sensor onboard ±0.1°C (TMP117), ±2% (MAX86141 SpO₂), ±10% (PMSA003I PM2.5). Kalibrasi factory + user calibration tiap 30 hari mitigate drift.
  • External validation: Belum ada multicentre RCT lokal. Roadmap: kolaborasi RS UI / RSCM / FK UGM dalam 2026-2027.
  • ML proxy: ml-client.js adalah rule-based proxy dikalibrasi ke output XGBoost/LightGBM. Untuk production-grade inference, butuh Cloud Functions backend (currently di Spark plan limit).
  • Privacy: Data tidak dijual. Default storage Firestore asia-southeast2. User control via Privacy page untuk opt-out research aggregate.
